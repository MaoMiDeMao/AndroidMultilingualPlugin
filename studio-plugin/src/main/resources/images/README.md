# 插件图片说明

本目录用于存放插件说明页面显示的截图。

## 需要的图片文件

请将以下截图保存到此目录（`studio-plugin/src/main/resources/images/`）：

1. **menu.png** - IDE 菜单截图
   - 显示 Tools -> 多语言助手 菜单选项
   - 建议尺寸：宽度 600-800px，高度适中

2. **generate-excel-dialog.png** - 生成 Excel 配置对话框
   - 显示"生成 Excel 配置"对话框的所有配置项
   - 建议尺寸：宽度 600-800px，高度适中

3. **generate-translations-dialog.png** - 生成翻译文件配置对话框
   - 显示"生成翻译文件配置"对话框的所有配置项
   - 建议尺寸：宽度 600-800px，高度适中

## 图片使用说明

这些图片会在以下位置显示：
- Android Studio 插件管理页面（Settings -> Plugins -> 已安装插件 -> 多语言助手）
- 插件详情对话框中的描述部分

## 图片格式要求

- **格式**：PNG 格式，保证清晰度
- **尺寸**：建议宽度 600-800px，高度根据内容自适应
- **文件大小**：建议每个图片小于 500KB，以保证加载速度
- **背景**：建议使用透明背景或与插件主题一致的背景色

## 注意事项

- 图片路径在 `plugin.xml` 中使用相对路径 `images/xxx.png`
- 图片会在插件打包时包含在 JAR 文件中
- 修改图片后需要重新编译和打包插件才能看到效果
